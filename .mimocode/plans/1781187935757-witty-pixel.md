# Wuthering Waves-Style Character Profile Redesign

## Overview
Redesign the about page from a 3-column RPG grid to a Wuthering Waves-inspired 2-column layout with elegant ornamental borders, gold accents, and glowing celestial effects.

---

## 1. Layout Restructure (`src/app/about/page.js`)

### New 2-column layout
- **Left column (~35%)**: Large portrait with ornate glowing frame, name, title, level badge
- **Right column (~65%)**: Stacked info panels — Stats, Bio, Certifications, Skills, Quest Log, Education, Research, Contact

### Structural changes
- Remove 3-column grid, replace with 2-column (portrait + info)
- Add decorative corner ornaments (CSS `::before`/`::after` with gradient borders)
- Portrait frame gets a gold + blue glow border with subtle animation
- Name displayed larger with elegant styling (not mono font)
- Level badge styled as an ornate crest/seal

### Files
- `src/app/about/page.js` — restructure JSX to 2-column layout
- `src/app/about/about.module.css` — complete restyle

---

## 2. Visual Styling (`src/app/about/about.module.css`)

### Card container
- Ornate border: thin gold (`#d4a853`) + blue gradient border
- Corner decorative elements via pseudo-elements
- Subtle inner glow matching celestial theme

### Portrait frame
- Taller aspect ratio (3:4.5 or similar)
- Gold + blue gradient border with glow
- Subtle animated shimmer effect on border

### Section panels
- Each section (Stats, Bio, etc.) in its own panel with:
  - Thin gold left border accent
  - Subtle background with slight gradient
  - Elegant section headers with gold accent color

### Stat bars
- Gold-to-blue gradient fill instead of pure blue
- Ornate bar design with subtle inner shadow

### Typography
- Section headers: gold color with text-shadow glow
- Body text: Inter font (already set as --font-sans)
- Labels: Fira Code for tech feel, smaller size

### Color additions to globals.css
- `--gold: #d4a853` — warm gold accent
- `--gold-glow: rgba(212, 168, 83, 0.3)` — gold glow effect

---

## 3. Responsive Behavior
- **Desktop (>1024px)**: 2-column, portrait left, info right
- **Tablet (768-1024px)**: 2-column with narrower portrait
- **Mobile (<768px)**: Single column, portrait centered on top, info below
